Markdown使用手册

这种东西应该最先写，免得每次写博客还要查markdown的相关文档。发表不发表看心情了，不过这只是一个手册，没办法代替教程。页面布局规范将来可能也会记录在这里。

标题语法

标题就是用#标记，最多有六级，一个#最大，六个#最小，记得后面加一个空格。

段落语法

要创建段落，请使用空白行将一行或多行文本进行分隔。

段落语法

如果不想分段，只想换行，在一行的末尾添加两个或多个空格，然后按回车键,即可创建一个换行。
如果觉得空格不明显，还可以用<br>显式换行，这样就不需要回车了。

转义字符

这个提前写，转义字符就是斜杠\，比如上面的<br>，可以写成\<br>这样就不会被markdown识别成换行了，
好像还有其他用法用的再说

强调语法

加粗用两个星号包围加粗部分（左右各有两个），例如：加粗，实际写法是**加粗**,转义时每个*都需要\修饰。

引用语法

要创建块引用，请在段落前添加一个 > 符号。例如

这是引用部分

块引用可以包含多个段落。为段落之间的空白行添加一个 > 符号。

实际写法是

> 这是引用部分

>

> 块引用可以包含多个段落。为段落之间的空白行添加一个 > 符号。

块引用可以嵌套。在要嵌套的段落前添加一个 » 符号（需要嵌套多少层就写多少>）。

列表语法

要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点。数字不必按数学顺序排列，但是列表应当以数字 1 起始。
要创建无序列表，请在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+) 。缩进一个或多个列表项可创建嵌套列表。

代码语法

要将单词或短语表示为代码，请将其包裹在反引号 (`) 中。
如果你要表示为代码的单词或短语中包含一个或多个反引号，则可以通过将单词或短语包裹在双反引号(``)中。

上面是创建短语代码，如果要创建代码块，使用围栏式代码块（fenced code blocks）
根据Markdown处理器或编辑器的不同，您将在代码块之前和之后的行上使用三个反引号（```）或三个波浪号（~~~）。
许多Markdown处理器都支持受围栏代码块的语法突出显示。使用此功能，您可以为编写代码的任何语言添加颜色突出显示。要添加语法突出显示，请在受防护的代码块之前的反引号旁边指定一种语言。例如

{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}

实际写法为

```json
{
"firstName": "John",
"lastName": "Smith",
"age": 25
}
```

分隔线语法

要创建分隔线，请在单独一行上使用三个破折号 (—) 并且不能包含其他内容。
